    Benutzerhandbuch – Robustes Portfolio V4.4

    📂 Bestehendes Portfolio importieren – zwei Wege

    Wenn Sie ein bestehendes Robustes Portfolio haben, können Sie es direkt in V4.4 laden. Es gibt zwei Wege – beide sind unabhängig voneinander:

    ✅ Weg 1 (empfohlen): Setup laden (.json)

    Der schnellste und zuverlässigste Weg. Die Datei portfolio_import.json enthält das vorbereitete Portfolio mit allen Fonds und Beträgen.

    1. Im Panel „Mein Portfolio" (rechts) auf das ⬆ Symbol (Pfeil nach oben) klicken
    2. Die Datei portfolio_import.json aus dem V4.4-Ordner auswählen
    3. Das Portfolio wird sofort mit allen Fonds und Beträgen geladen

    ⚠ Kein separater Upload der PDF-Datei nötig! Die JSON-Datei enthält bereits alle Informationen.

    ⚡ Weg 2 (automatisch, wenn PDF maschinenlesbar): Portfolio laden (PDF)

    Über den Button „Portfolio laden (PDF)" im Header können Sie direkt eine Depotauszug-PDF hochladen. Das System versucht, WKN-Nummern und Kurswerte automatisch auszulesen.

    1. „Portfolio laden (PDF)" Button im Header anklicken
    2. Die Depot-PDF auswählen (muss maschinenlesbaren Text enthalten)
    3. Fonds werden automatisch dem Turm zugeordnet und Beträge eingetragen

    ⚠ Scan-PDFs (Fotos/Scans ohne Textebene) können nicht direkt gelesen werden – bitte Weg 1 verwenden.

    🖥️ Betriebssystem-Kompatibilität

    Das Dashboard ist eine reine Webanwendung und läuft in jedem modernen Browser. Es ist kein Betriebssystem und keine Installation erforderlich:

    SystemUnterstützt?Hinweis
    🍎 macOS✅ VollständigChrome, Safari, Firefox
    🪟 Windows✅ VollständigChrome, Edge, Firefox
    🐧 Linux✅ VollständigChrome, Firefox
    📱 Tablet / iPad⚠ EingeschränktLayout für Desktop optimiert

    💡 Alle Daten werden lokal im Browser gespeichert (LocalStorage). Es werden keine Daten ins Internet übertragen.

    🆕 Portfolio neu anlegen (ohne Importdatei)

    1. Gesamtvermögen eingeben – Im Bereich „Anlageplanung" den Betrag eingeben.
    2. Schicht auswählen – Auf eine farbige Schicht im Turm klicken, um die Fondsliste zu öffnen.
    3. Betrag und Fonds wählen – Einmalbetrag / Sparrate eintragen, dann + Auswählen klicken.
    4. Portfolio verwalten – Im Panel rechts alle gewählten Fonds sehen; mit ✕ entfernen.
    5. Speichern – Mit dem ⬇ Pfeil-Symbol als JSON sichern, um das Setup jederzeit wieder laden zu können.

    📋 Schichten-Reihenfolge im Turm (unten → oben)

    1. Tagesgeld
    2. Kapitalreservefonds (Kasse / Wertsicherung)
    3. Defensive Vermögensverwalter
    4. Ausgewogene Vermögensverwalter
    5. Dynamische Vermögensverwalter
    6. Märkte (Enges Benchmarking auf Aktien)
    7. Spezialitäten / Themen

    🏷️ Jede Schicht hat eine Sammelposition „[Schicht] – Fonds delistet" für Altpositionen, die inzwischen von der VEM-Liste entfernt wurden, aber noch im Depot liegen.
